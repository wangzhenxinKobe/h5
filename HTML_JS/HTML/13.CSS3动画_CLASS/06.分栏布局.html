江疏影，1986年9月1日出生于上海[1-2] ，中国内地女演员，毕业于上海戏剧学院2004级表演系本科班，英国东英格利亚大学传媒经济学2009级硕士。

2005年出演首部电视剧《飞花如蝶》“金凤”一角而出道。[3] 2013年在电影《致我们终将逝去的青春》出演校园女神“阮莞”得到广泛关注，[4] 凭借该片获得第五届英国万像国际华语电影节最佳女配角奖和第八届亚洲电影大奖最佳新演员奖。[5-7] 2014年在都市时尚情感剧《一仆二主》饰演绿茶女神“顾菁菁”，凭借该片提名第21届上海电视节最佳女配角，斩获第17届华鼎奖中国百强电视剧最佳女配角。[8-15] 2015年初在青春励志剧《长大》饰演萌妹天使“陈曦"。[16-17] 同年8月出演都市浪漫爱情偶像剧《最佳前男友》女一号强势女王“傅方思”。[18] 2016年4月1日，主演青春热血励志剧《旋风十一人》，剧中饰演表格领队“裴朵”[19-20] 。2016年5月31日，主演的都市情感剧《好先生》首播，剧中饰演千金烈女“江莱”。

2016年7月7日，主演都市爱情剧《一路繁花相送》，剧中饰演女一号“辛辰”[21] 。
